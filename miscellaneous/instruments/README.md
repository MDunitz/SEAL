# Instruments

Instrument-specific documentation: manuals, configuration files, explanatory diagrams, maintenance/care details, and calibration records.

## Suggested Structure

```
instruments/
├── INSTRUMENT_NAME/
│   ├── README.md          # Quick reference: model, location, key settings
│   ├── manual.pdf         # Or link to online manual
│   ├── calibration/       # Calibration records and schedules
│   └── diagrams/          # Schematics, flow diagrams
```
